# HTML Test Archive

This directory contains historical HTML verification files used during UI development.

## Contents

Archived verification files for:
- Authentication UI
- Warm styling implementations
- Tab components (Circles, Contacts, Groups, Tags)
- Page layouts (Edits, Preferences, Suggestions)
- UI components (Modals, Toast notifications, Floating chat)
- Responsive layouts and navigation

## Active Tests

For current test files, see:
- **Active HTML Tests**: `/tests/html/`
- **Integration Tests**: `/tests/integration/`
- **Test Documentation**: `/docs/testing/`

## Purpose

These files are kept for historical reference and to understand the evolution of the UI design system.
